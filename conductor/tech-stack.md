# Technology Stack

## Core Technologies
- **Language:** Dart
- **Framework:** Flutter (SDK ^3.10.1)
- **Backend as a Service:** Supabase (`supabase_flutter`)
- **State Management:** GetX (`get`)
- **Local Storage:** GetStorage (`get_storage`)

## UI & UX Components
- **Form Handling:** `flutter_form_builder`, `form_builder_validators`
- **Responsive Utilities:** `flutter_screenutil`
- **Navigation & Routing:** GetX
- **Animations & Feedback:** `shimmer` (Loading skeletons)
- **Image Management:** `cached_network_image`
- **Lists & Pagination:** `infinite_scroll_pagination`
- **Overlays:** `modal_bottom_sheet`

## Infrastructure & Services
- **Authentication:** Supabase Auth + Biometric (`local_auth`)
- **Environment Management:** `flutter_dotenv`
- **Connectivity:** `connectivity_plus`
- **System Services:** `url_launcher`, `path_provider` (inferred)
- **Database (Client-side):** `sqflite` (inferred)
